# Sujatha's Essentials - Backend

## Setup
1. Copy `.env.example` to `.env` and fill values.
2. `npm install`
3. `npm run dev` (development)
4. Seed sample data: `npm run seed`

## Features
- JWT Auth
- Product, Cart, Order models
- Tours endpoints
- Cloudinary image uploads
- Nodemailer email helper
- Validation via Joi
